QuickDeploy - Deployment Guide

Deploy your application in minutes with our CLI tool.

Installation

npm install -g quickdeploy-cli

Quick Start

1. Initialize your project:

quickdeploy init

2. Configure your deployment:

quickdeploy config set region us-east-1

3. Deploy:

quickdeploy push

Environment Variables

Set your deployment secrets:

quickdeploy secrets set DATABASE_URL "postgres://..."
quickdeploy secrets set API_KEY "your-key"

Monitoring

View deployment status:

quickdeploy status
quickdeploy logs --tail

Rollback

If something goes wrong:

quickdeploy rollback
