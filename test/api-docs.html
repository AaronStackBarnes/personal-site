PayFlow API Documentation

Authentication

All requests require an API key in the header:

Authorization: Bearer YOUR_API_KEY

Base URL

All API requests should be made to:

https://api.payflow.com/v2

Create a Payment

To create a new payment, make a POST request:

POST /payments
{
  "amount": 1000,
  "currency": "usd",
  "description": "Order #1234"
}

Response

{
  "id": "pay_1234567890",
  "status": "succeeded",
  "amount": 1000
}

Error Handling

The API returns standard HTTP status codes:

  • 200 - Success
  • 400 - Bad request
  • 401 - Unauthorized
  • 500 - Server error
